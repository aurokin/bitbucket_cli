# Roadmap

## Next Commands

- Pipeline actions and logs
  Extend the current pipeline list and view support with supported follow-up actions like log inspection and rerun or stop flows only where Bitbucket Cloud has real API support.

## Later Phase

- Browse redesign
  Add `bb browse` with `gh`-style default browser behavior while still supporting an easy URL-printing mode for agents and scripts.

- Browser login
  Add OAuth-based browser auth after the current API-token-first command set is more complete.
